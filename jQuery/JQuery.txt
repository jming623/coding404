JQuery: 자바스크립트 언어를 간편하게 사용할 수 있도록 단순화시킨 오픈소스 기반의 자바스크립트 라이브러리

JQuery의특징및 장점
1.중복되는 기능과 코드가 엉키는 것을 피하고 이미 개발된 많은 플러그인을 쉽고 빠르게 이용 가능하다.
2.크로스 브라우저를 사용할 수 있는 특징을 가지고 있으며 각 브라우저 마다 다르게 발생하는 이벤트, 객체등을 처리하고자
여러개의 메소드를 따로 사용하는 것이 아니고 jQuery에서 제공되는 메소드로 간단하게 해결할 수있습니다.
3.JQuery라이브러리는 JQuery라는 이름의 객체로 생성된다.
4.선택자를 이용하여 적용할 요소를 선택하고 선택자는 CSS선택자 및 JQuery선택자를 이용한다.

기본적으로 $로시작해서 ()소괄호안에 ""쌍따옴표를 넣어 사용한다.

$("*") 모든요소 선택
$(this) 현재요소 선택
$("p.intro") = 모든p태그에 클래스명이 intro인 요소선택
$("p:first") = 첫번째 p태그 선택
$("ul li:first") = 첫번째 ul태그 안에있는 첫번째 li태그선택
$("ul li:first-child") = 모든 ul태그안에 첫번째 li태그들을 선택
$("[href]") = href속성을 가진 모든 요소선택
$("a[target='_blank']") = 모든 a태그안에 target속성값이 _blank인 모든요소 선택
$("a[target!='_blank']") = 모든 a태그안에 target속성값이 _blank가 아닌 요소들을 선택
$(":button") = 모든 버튼속성을 가진 요소들을 선택(button태그, type이 button인 input태그)
$("tr:even") = 모든 tr요소를 가져와서 짝수 인덱스 번호를 가진 요소들만 선택 (0부터시작)
$("tr:even") = 모든 tr요소를 가져오서 홀수 인덱스 번호를 가진 요소들만 선택 

필터링에 사용 가능한 선택자
:eq(n) = 선택한 요소중에서 인덱스가 n인 요소 선택
:gt(n) = 선택한 요소중에서 인덱스가 n보다 큰(높은)요소들을 선택
:lt(n) = 선택한 요소중에서 인덱스가 n보다 작은요소들을 선택
:even = 선택한 요소중에 인덱스가 짝수인 요소들을 선택
:odd = 선택한 요소중에 인덱스가 홀수인 요소들을 선택
:first  = 선택한 요소중에 첫번째 요소 선택
:last = 선택한 요소중에 마지막 요소 선택
:animated = 선택한 요소중에 애니메이션 효과가 실행중인 요소들 선택
:header = 선택한 요소중에 h1~h6까지의 속성을 가지고있는 요소들 선택
:lang(언어) = 선택한 요소중 (언어)로 이루어진 요소들을 모두선택
:not(선택자) = 선택한 요소중 선택자에 들어가는 요소와 일치하지 않는 요소들을 선택
:root = 선택한 요소중에서 최상위 루트 요소를 선택
:contains(텍스트) = 선택한 요소중 (텍스트)를 포함하는 요소 선택

:checked = checkbox나 radio에서 선택된 요소선택
:selected = <option>에서 선택된 요소선택
:focus = 포커스되어있는 요소선택
:disabled = 비활성화 되어있는 요소선택
:enabled = 활성화되어있는 요소선택
